Business Model Canvas

Justificativas

Grande roblema ambiental da região, enchentes

Produtos

Previsor de enchente

Stakholders Externos

IDR Paraná;
Defesa Civil;
Sinepar;
Instituto Nacional de Meteorologia

Premissas

Auxiliar as autoridades locais a reagir às enchentes, buscando reduzir os danos causados

Riscos

Danos ao dispositivo, furto, problemas com a rede.

Objetivos

Alertar a população e as autoridades para tomar as devidas providências ao nível que sejam capazes.

Requisitos

Dados meteorologicos, permissão ambiental para instalação do prototico e acesso a rede.

Equipe

Ianoski: Dev. Fullstack - Web, Manutenção de Redes
Perek: Estatistica, Contadoria, Engenheiro Mecatronico
Theo: Relações, Marketing e Publicidade, Legislação, Vendas.

Grupos de Entrega

1. Desenvolver a ideia do prototipo
2. Desenvolver o prototipo
3. Buscar por informações metereológicas
4. Montar os cálculos
5. Desenvolver a API
6. Integração web

Linha do tempo

1. Levantamento de dados - 2 semanas
2. Desenvolvimento do prototipo - 3 semanas
3. Testes do prototipo - 1 mes
4. Desenvolvimento da API - 1 meses
5. Integração com o site - 1 mes
6. Testes finais - 1 semana

Benefícios

Monitorar o rio, para advertir sobre os riscos de possiveis mudanças nele.

Restrições

1. Produção em massa
2. Impedir enchentes
3. Rede
4. Energia
